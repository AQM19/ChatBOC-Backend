# ChatBOC

## Uso
Para la inicialización del proyecto se deberán seguir los siguientes pasos:

- Desactivar conda (si está previamente activado)

    ``` bash
    conda deactivate
    ```
- Activar el entorno virtual local
    ``` bash
    source chatboc-env/bin/activate
    ```
